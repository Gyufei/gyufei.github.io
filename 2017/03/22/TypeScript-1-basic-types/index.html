元组

Null && Undefined

在 Ts 中为两种类型，这两种类型是其他所有类型的子类型，可以赋值给任意类型(除了never类型。

使用--strictNullChecks标记，则 null 和 undefined 只能赋值给自身或赋值给 void

Void

表示无任何类型，通常用于无返回值函数的返回值定义，只能被赋值undefined和null。

Any

  • any表示任意类型，可以赋任意值。
  • 与Object类型不同的是，any也可以调用任意方法。
  • 可以用来声明一个包含不同类型的数组, 如let arr: any[] = [1, '2', true]

Never

表示永不存在的值类型，如总是抛出异常的函数的返回值, 或者永不为真的变量值等。
never是任何类型的子类型，可以赋值给任意类型。
never不能被任意类型赋值，除了自身。即便any也不行。
